# AI Skills

Hi everyone 👋

For a while now, I’ve been using AI to plan features before I build them.  
At some point, while working on a side project, I noticed something, and it's that when I describe tasks using Gherkin scenarios, the AI gets way better at breaking work into solid technical TODOs.

So this repo is my little playground for that approach:

- custom skills and instructions,
- feature planning with Gherkin-style tasks,
- and prompts/workflows that can turn those plans into GitHub issues with `gh cli`.

It’s basically:  
“Plan smart first, code second, panic less.”  
(No promises about *zero* panic.)
